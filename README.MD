# Boostcamp Backend Study

벡엔드 학습 과정에서는 데이터베이스와 연동되는 REST API를 개발하는데 필요한 기술을 다룬다. 아래와 같은 기술 요소를 사용할 예정이다.


## Use

•개발언어 : Java

•IDE : IntelliJ

•빌드 : Maven

•테스팅 : JUnit

•IoC Container : Spring Framework

•웹어플리케이션 서버 : Spring Boot + Embedded Tomcat

•로깅 : Slf4 + Logback

•DB 입출력 : Spring JDBC

•REST API : Spring MVC
